Research Experience

Continual Learning in Neural Networks

Bachelor's Thesis, IIT Bombay
Guide : Prof. Sunita Sarawagi

Current neural networks training algorithms face the problem of catastrophic forgetting when dataset is provided incrementally. However, this is the case when a new word needs to be learnt in a language translation system or a new face has to be recognized in face recognition system. Training from scratch using complete dataset is infeasible. We are working using distillation and sampling useful points from previous dataset to be used with new dataset while training to learn new task and retain knowledge of previous tasks.

Dynamic Memory Management for Training Deep Neural Networks

R&D Project, IIT Bombay
Guide : Prof. Purushottam Kulkarni

Currently, training of state-of-the art neural networks take up space on the order of tens of GBs, whereas the size of GPU DRAM memory is limited (about 12 GB). This limits the batch size used for training or the network architecture. Although multiple GPUs could be used for training with large batch-sizes, our aim was to train the models using a single GPU. We improved one of the previous works which used comparatively large system (CPU) memory in addition to GPU DRAM and transferred data across them while training the network. We removed synchronization at the end of each layer of network, proposed heuristic to avoid memory fragmentation and showed the improvement in performance of our design. We also showed feasibility of compression of data with CPU cores to reduce memory consumption. Our work has been accepted at International Parallel and Distributed Processing Symposium (IPDPS), 2019

Local Max-Cut in Smoothed Polynomial Time

Research Internship, EPFL, Switzerland
Guide : Prof. Ola Svensson

There exist weighted-graphs for which FLIP takes exponential number of steps, while empirically, it has been found that for most of the weighted-graphs, the algorithm converges reasonably quickly. Smoothed analysis attempted to find if FLIP converged in polynomial time on expectation when the weights of the graph are added with a small amount of random noise. Extending the ideas used in the previous works for complete graphs, we were able to prove smoothed-polynomial complexity for a special kind of graphs - a combination of complete and bounded degree graphs.

Refinforcement Learning

We applied Synaptic Intelligence – used to avoid catastrophic forgetting in neural networks – to batch-RL helping to retain knowledge of experiences obtained from previous batches. A neural network to represent Q-value was trained using batch-RL and experience replay, regularized with Synaptic Intelligence. This reduced the amount of experience per batch required and improved the rate of convergence during training.

Flow-Based Image Abstraction

Digital Image Processing

We used line drawing, region smoothing and color quantization to convert natural images to cartoon-like images. We implemented the paper on the same title. As a first step, we found a vector field – Edge Tangent Flow (ETF) – which aligns along edges in the image. Rationalizing Neural Predictions

Machine Learning

The task was to predict the ratings a user would have given to a product based on the textual review written by him on that product. The model also has to justify its prediction by providing excerpts from the textual review. We first tried a simple method involving attention mechanism, using attention weights as a measure of the focus given by the model for a particular text while prediction. Then we implemented a paper on the same title, which used a Generator-Encoder model. The Generator gives the probability distribution of excerpts and the encoder predicts the ratings. These are trained end-to-end by feeding samples from Generator to Encoder and optimizing for predicted value.

Intelligent Pacman Agent

Artificial Intelligence

I worked on designing a Pacman agent to avoid ghosts and eat food dots as fast as possible. As a first algorithm, I implemented genetic algorithm which learnt policy function with neural network using concepts of survival of the fittest, cross breeding and mutation. Then, I implemented algorithms and heuristics such as Expectiminimax with alpha-beta pruning, Q-learning. This improved the performance and time required for training compared to genetic algorithm. I then worked in a setting where ghosts are invisible and used Particle Filters and Dynamic Bayesian Nets to track them, using radar outputs.

Network of ATM Controllers

Digitial Logic Design

We designed an ATM machine on FPGA with database backend on PC, which were connected through USB communication. We designed state machine for encryption and communication modules of ATM machine. Then, we simulated multiple banks using different FPGAs connected to their own backend PCs. These FPGAs were connected through Ethernet to allow accessing of some bank account from any ATM. We designed protocols and state machine for communication over Ethernet, avoiding link failures as well. All these state machines were implemented on VHDL.
